# SDL keyboard/mouse default controls

The SDL desktop keyboard/mouse fallback exposes a fake controller for the console input stack, plus keyboard-only semantic input points for gameplay mappings. Native SDL gamepads keep the original console input point names and mappings.

## On foot

- Move: `W`/`A`/`S`/`D`
- Camera/look: mouse; optional keyboard look fallback: `I`/`J`/`K`/`L`
- Jump: `Space`
- Sprint/dash: `Left Shift` / `Right Shift`
- Interact / enter vehicle: `E`
- Attack/kick: left mouse, `F`, or `Ctrl`
- Pause: `P` / `Pause`
- Back/cancel in menus: `Esc` / `Backspace`
- Camera toggle: `Q`
- First-person toggle: `C`

## Driving

- Accelerate: `W`
- Brake/reverse: `S`
- Steer: `A` / `D`
- Handbrake: `Space`
- Horn: `H` or `Shift`
- Reset car: `R` / `Backspace`
- Camera/look: mouse; optional keyboard look fallback: `I`/`J`/`K`/`L`

## Menus

- Navigate: arrow keys (WASD may still move one item through the left-stick fallback)
- Confirm: `Enter` / `Space`
- Back/cancel: `Esc` / `Backspace`
